 A simple terminal-based Python tool for Geographical Information from IP/DNS

A fun little tool that make use of a free IP API to give Geographical information along with Google coordinates

Usage:

	Prompts users to type in an IP-adress or DNS(Website name) which returns Geo location for said IP or DNS.

	help or --help for more information
